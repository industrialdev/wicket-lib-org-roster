# Cascade Strategy: Config Schema

Cascade mode uses shared config keys rather than a dedicated `cascade` namespace.

## Most Relevant Keys

- `roster.strategy = cascade`
- `feature_flags.membership_resolution_prefer_current_cycle`
- `member_addition.*`
- `permissions.*`
- `relationships.*`
- `ui.member_list.*`
- `ui.member_view.*`
- `bulk_upload.*`
